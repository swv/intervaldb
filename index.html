intervaldb by cmungall

prolog wrapper for PyGR Nested Containment Lists (NCLs)

An SWI-Prolog wrapper for the PyGR NCL library, allowing for faster retrieval of genomic features from large intervals.

Dependencies

PyGR

Install

First you will need to build libnclist.a * Download PyGR from http://code.google.com/p/pygr/ == cd pygr make cp libnclist.a ../intervaldb/ == Then run the normal configure/make/make install cycle: == ./configure make make install ==
